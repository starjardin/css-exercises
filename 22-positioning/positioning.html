Positioning

  1. The default value for position is static
  2. Some of the properties you need to supply when using any non-deault position value are: top, left, bottom, and right.
  3. The inferred display value of an element has been positioned is: the value display: block.
  4. One of the non-default values for position still maintains a box in the flow document, that is relative
  5. Absolute positioned layouts are useful for elements that should not interact.
  6. Css' s Stacking Content is a three dimensional conceptualization of html along an imaginary Z-axis relative to user.

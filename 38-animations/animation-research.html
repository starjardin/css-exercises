Animation.

Research about animation

  1. Keyframe animaton setup needs some unique code, it start with @Keyframe
  2. What makes the Keyframes code diferent to the rest of CSS is that we have to declare it first.
  3. We use a Keyframe animation we set up earlier as a property in a ruleset for an element is by calling their name and aniamation-duration.
  4. Keyframe aniamations have other keywords that aren't available in transitions, these keywords are:
    • animation-play-state
    • animation-fill-mode
    • aniamtion-iteration-count
    • animaation-direction
    • alternate
    • infinite
    • reverse
    • alternate-reverse
    • running
    • paused
  5. We can animate as many as we want as long as the properties are animatable.
  6. These properties are animatable:
    • background
    • border, box-model
    • color
    • column
    • font
    • margin
    • scroll
    • visibility

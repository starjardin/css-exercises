Float and clear

  • Their default value are:
    • Float: none
    • Clear: none
  • The possible values for clear are:
    • Left
    • Right
    • None
    • Both
  • The rules associated with a floated item in terms of order is the floated must comes first before that it floating with. when an element float it become inline block.
  • Yes, we can float other elements besides images.
  • Yes, there is a way to create a two-column layout with floats.

Pseudo element

  1. For a pseudo element to work we need a special property set for the element, that property is the content property
  2. We could leave the content property empty and just treat the Pseudo-element. However we can't remove the content property, at the very least, the content property needs double quotes as it value
  3. The various pseudo element are:
    • font-prorerteis
    • color properties
    • background properties
    • word spacing
    • letter spacing
    • text decoration
    • verical-aling
    • text-tranform
  4. The reason why would we use a pseudo element is that we do not have a specific selectors to work with, so the pseudo element can help use to style our document.
    A CSS pseudo element is used to style specified pparst of an element.

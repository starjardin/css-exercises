CSS Inheritance.

Answering the qwestions

  1. When we use CSS inheritance, we mean to inherit the parent's value to its children. That means the children will have the same value as their parents if we don't specify their value.
  2. It affects the curent values to its descendent.
  3. Here are some exemple of properties have values that are inherited by default: color, font, font-family, font-size, font-style, font-variant, font-weight, letter-spacing, line-height, text-align, text-indent, text-transform, visibility.
  4. Using revert , you will put back those styles while unset will not. unset will simply set the margin to initial, The initial keyword represents the specified value that is designated as the property's initial value. Thus, its meaning depends on the property, but not on anything else.

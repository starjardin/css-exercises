Learning grid

  1. By default, grid has one column.
  2. The difference between grid and flexbox are:
    • Flexbox was design for one dimmension either row or column
    • Grid was design for two demrnsions row and column at the same time
  3. If the items should be arranged in single dimension row or coluumn, it better to use flexbox.
  4. If the items should be arranged in two clolumn at the same time either row and column it would be nice to use grid
    5. More concsie way to write out the following grid-temlate-columns: 10rem 10rem 10rem 10rem;is grid-temlate-columns: repeat(4, 10rem);
  5. We can specify a minimum height for a grid row, but still allow the row to increase in height if its content id too tall by setting the property: grid-template-rows: minmax(min-content, auto;

Box model.

Drawing box model.

Box Model is a container of every visible html element. It is used to create the desing and layout of web pages.

Box model is a box in every visible html that contains its content. A box model has: margins, paddings, borders, and a content. Margins are the lines which coverd the box, borers are the part that follwed after the margins, paddings are the part which cover the content box, and the content box is the box which contains the content of the html. Those four elements together gives the term box model. You can specify each one of those propreties.

Box-model is a combination of content-box, padding-box, border-box, and margin-box.

Definition lists

Content-box
Content-box is the deepest part of the box which contains the content of an html element
Padding-box
Padding-box is the second insid part of the box which is in between the content-box and the border box, in other words the padding box separates the content-box and the border box
Border-box
Border box is the part arround the padding box that is in between the margins and the padding box
Margin-box
Margin-box serrounds the boreder-box.
